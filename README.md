# 🏏 IPL Analysis Dashboard (2008–2025) | Power BI

## 📌 Project Overview
This Power BI dashboard provides a comprehensive analysis of IPL seasons from 2008 to 2025. It includes team performance, player statistics, and season-wise insights.

---

## 🎯 Objectives
- Identify season champions and runners-up
- Analyze total sixes, fours, matches, teams, and venues
- Display Orange Cap and Purple Cap statistics
- Compare teams using points table
- Track player performance metrics

---

## 📊 Key Features
- Dynamic Season Filter
- KPI Cards (Matches, Sixes, Fours, Venues)
- Orange & Purple Cap Insights
- Points Table Breakdown
- Team & Player Performance Cards
